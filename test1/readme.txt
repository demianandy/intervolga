1. news.php. В этом файле находится массив с новостями.
Там же находится уловие, согласно которому выбирается новость из этого массива (по умолчанию или при выборе темы пользователем).

2. data.php. Файл обращается к news.php и формирует заданные переменные ($a - полный текст новости, $link - ссылка на страницу с полным текстом новости, $b - обрезанный фрагмент новости размером в 180 символов).

3. test1.php. Файл обращается к data.php и выводит фрагменты новости с 3 вариантами открытия ее полного текста:

3.1 link1.php - обычная ссылка, которая берет полный текст новости оттуда же, откуда и сам родительский файл test1.php, - из data.php.

3.2 link2.php - полный текст новости берется из сессии, которая запускается в родительском файле - test1.php.

3.3 link3.php - полный текст новости берется из GET-запроса, который формируется в ссылке родительского файла test1.php.
